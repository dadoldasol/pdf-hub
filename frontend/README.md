# PDF Knowledge Hub Frontend

Dependency-free frontend MVP for the PDF Knowledge Hub.

## Run

From the project root:

```powershell
node frontend\server.js
```

Open:

```text
http://127.0.0.1:5173
```

Backend default:

```text
http://127.0.0.1:8000
```

If the frontend was already running, stop it with `Ctrl+C` in that terminal and run `node frontend\server.js` again.
